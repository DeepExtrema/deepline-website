# 5) main.js — add/adjust logic

## 5.1 Reveal-on-scroll

```js
const io = new IntersectionObserver((entries)=>{
  entries.forEach(e=>{
    if(e.isIntersecting){ e.target.classList.add('reveal'); io.unobserve(e.target); }
  });
},{ threshold:0.2 });

document.querySelectorAll('.section, .card').forEach(el=>{
  el.classList.add('pre-reveal'); io.observe(el);
});
```

## 5.2 Auto-hide nav (retain if already implemented)

Ensure header gets `.is-scrolling` class when scrolling down ~24px and removes on scroll-up/top.

## 5.3 Analytics events (no vendor lock; console fallback)

```js
document.querySelectorAll('[data-event]').forEach(el=>{
  el.addEventListener('click', ()=>{
    if (window.posthog?.capture) posthog.capture(el.dataset.event);
    else if (window.gtag) gtag('event', el.dataset.event);
    else console.log('event:', el.dataset.event);
  });
});
```

Events to ensure exist on elements:

* `cta_nav_primary`
* `cta_hero_primary`
* `cta_hero_calendly`
* `cta_mid_how`
* `cta_mid_playbooks`
* `playbook_cta_*` (per card)
* `form_submit_attempt` / `form_submit_success`

## 5.4 Form validation

```js
const form = document.querySelector('#early-access-form');
if(form){
  form.addEventListener('submit', (e)=>{
    const email = form.querySelector('input[type="email"]')?.value.trim();
    const role  = form.querySelector('select[name="role"]')?.value;
    const pain  = form.querySelector('#pain')?.value.trim() || '';
    if(!email || !role){
      e.preventDefault(); alert('Email and role are required.'); 
      document.body.dispatchEvent(new CustomEvent('form_submit_attempt'));
      return;
    }
    if(pain && pain.length < 10){
      e.preventDefault(); alert('Tell us your pain in ~10+ chars.');
      return;
    }
    document.body.dispatchEvent(new CustomEvent('form_submit_success'));
  });
}
```
